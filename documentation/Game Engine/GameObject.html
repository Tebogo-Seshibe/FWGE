GameObject

FWGE.Game

The main object container for object types.

ID : String

Unique identifier for the gameobject

Transform : Transform

The transform object attached to the current gameobject

Children : Array

An array of gameobjects. All children transformation will be relative to the parent gameobject.

RenderMaterial : RenderMaterial

The render material attached to this gameobject.

Mesh : Mesh

The mesh attached to this gameobject.

LightItem : LightItem

The light item attached to this gameobject.

PhysicsItem : PhysicsItem

The physics item attached to this gameobject.

Animation : Animation

The animation attached to this gameobject.

particlesystem : ParticleSystem

The particle system attached to this gameobject.

Begin : Function

This method is called upon object creation.

Update : Function

This method is called after each render frame

End : Function

This method is called once the gameobject if destroyed.

AddChild(gameobject : undefined) : undefined

Pushes a gameobect to the current object's childrens array, and move it down the rendering tree.

RemoveChild(gameobject : undefined) : undefined

Removes a gameobject from the current object's childrens array, and moves it up the rendering tree.

Clone(gameobject : undefined) : undefined

Creates a clone of a gameobject. If no gameobject is provided, it creates a clone of the calling gameobject.

Destroy(timeout : undefined) : undefined

Destroys the object after a given amount of time

ObjectUpdate(param : undefined) : undefined

Updates the object
